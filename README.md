# Lyra Voice

**Local-first voice dictation for macOS** — press a hotkey, speak, get clean formatted text pasted into any app.

Lyra Voice — macOS-приложение для голосовой диктовки: запись по горячей клавише, локальная
транскрибация через [whisper.cpp](https://github.com/ggerganov/whisper.cpp), умная постобработка
текста (пунктуация, абзацы, словарь, голосовые команды) и автоматическая вставка результата в
активное приложение.

## Возможности

- **Горячие клавиши**: переключение запись/стоп и режим push-to-talk — независимо друг от друга.
- **Локальное распознавание** через `whisper.cpp` с тёплым сервером в памяти (без перезагрузки модели
  на каждую диктовку); выбор модели `tiny … large-v3` под баланс скорости/точности.
- **Облачное распознавание** (OpenAI, опционально, по своему API-ключу) как явный выбор пользователя.
- **Постобработка текста**:
  - правило-based чистка (всегда, мгновенно, офлайн): пунктуация, регистр, голосовые команды
    («новый абзац», «тире», «открыть кавычки» и т.п.), словарь замен, удаление слов-паразитов;
  - умная разбивка длинного текста на абзацы по смыслу;
  - опциональная локальная LLM-полировка («Умная (ИИ)») через `llama.cpp` — грамматика, самоисправления,
    форматирование без искажения смысла.
- **Умный контекст экрана** (Accessibility): подстройка форматирования под активное поле — email,
  чат, код, терминал, поиск, документ.
- **Оверлей записи и обработки**: компактная индикация состояния поверх остальных окон.
- **История диктовок** и статистика использования.
- **Управление медиа**: пауза/приглушение звука на время записи.
- **Интерфейс на русском и английском** (`L.t`), переключается в настройках.

## Установка

Требуется **Apple Silicon Mac** (M1 и новее), macOS 13+.

1. Скачайте `LyraVoice-vX.Y.Z.dmg` со страницы [Releases](../../releases).
2. Откройте DMG и перетащите **Lyra Voice** в папку **Applications**.
3. Первый запуск (приложение подписано самоподписанным сертификатом, не нотаризовано Apple):
   - **правый клик** по «Lyra Voice» в Applications → **Открыть** → в диалоге ещё раз **Открыть**;
   - либо разрешите в System Settings → Privacy & Security → **Open Anyway**;
   - если macOS пишет, что приложение «повреждено», выполните в Терминале один раз:
     ```sh
     xattr -cr "/Applications/Lyra Voice.app"
     ```
     (снимает карантин со скачанного бандла) и откройте снова.

Приложение **самодостаточно**: движки распознавания (`whisper.cpp`) и полировки (`llama.cpp`)
вшиты внутрь — **homebrew или другие установки не нужны**. При первом запуске оно запросит доступ
к микрофону и Accessibility (для вставки текста) и предложит скачать нужные модели (whisper +
модель ИИ для режима «Умная (ИИ)») — это единственное, что качается отдельно из-за их размера.

## Сборка из исходников

Требования: macOS (Apple Silicon), Xcode/Swift toolchain, [whisper.cpp](https://github.com/ggerganov/whisper.cpp)
и [llama.cpp](https://github.com/ggerganov/llama.cpp) (для dev-запуска; в релизный `.app` они вшиваются).

```sh
swift run LyraVoiceCoreSmokeTests   # тесты core-логики
swift build                          # сборка
scripts/download-models.sh           # скачать GGML-модели whisper.cpp
scripts/run-dev-app.sh               # dev: пересобрать .app и перезапустить (движки из homebrew)
scripts/make-dmg.sh                  # релиз: самодостаточный .app + DMG (движки вшиваются в бандл)
```

Подробнее о структуре проекта и интерфейсе — в [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md).
